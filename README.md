# Hash-Test

Test the Performance of different Hash-Functions.

## Rerequisites

Install Docker on your System.

## Build

```bash
$ docker build . -t hash-test
```

## Install & Run

```bash
$ docker run -ti --rm hash-test bash
# make install
# make
```
